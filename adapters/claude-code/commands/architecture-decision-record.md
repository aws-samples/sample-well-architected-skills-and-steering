Run wa-builder in **ADR mode** to create or revise an Architecture Decision Record.

Follow the wa-builder skill, "Mode: Architecture Decision Record" section. Analyze the codebase to understand current state, evaluate options with code evidence, assess WA pillar impact, and produce a structured ADR with trade-offs, migration path, and review triggers.
